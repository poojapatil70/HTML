Key Computer Science Terms

Algorithm
A step-by-step procedure to solve a problem or perform a computation.
Database
An organized collection of structured information or data,typically stored electronically.
Compiler
A program that translates source code written in one programming language into another language.
APL
Application programming Interface, a set of functions and protocols for building software applications.
